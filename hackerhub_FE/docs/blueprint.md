# **App Name**: HackHub Explorer

## Core Features:

- Hackathon Scraping: Scrape hackathon data from MLH and Devpost using the provided API endpoints.
- Search and Filter: Implement advanced search and filtering options based on platform, domain, and location.
- Results Display: Display scraped hackathon events in a responsive and visually appealing card grid layout.
- Event Details: Show full hackathon event details in a modal or dedicated page, rendering HTML safely.
- Favorites Management: Allow users to add hackathons to their favorites and persist them across sessions using localStorage.

## Style Guidelines:

- Color Palette: Dark scheme, embodying technology and innovation, primary color of intense purple (#9D32D3) complemented by desaturated purple background, accent of slightly more saturated and lighter violet.
- Primary color: Intense purple (#9D32D3) for boldness and forward-thinking appeal.
- Background color: Soft, desaturated purple (#262329).
- Accent color: Lighter and slightly more saturated violet (#C774E8) for CTAs.
- Font: 'Inter', a grotesque-style sans-serif font that conveys a modern and neutral aesthetic suitable for headlines or body text
- Use Lucide React icons, ensuring consistent and modern styling.
- Employ a mobile-first approach with a responsive card grid layout.
- Incorporate subtle animations and hover effects to enhance user engagement.
